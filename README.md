# e2e-puppeteer
前端端到端测试实践

### 环境
```
node: 12.14.0 
```

### 解决问题
通过puppeteer模拟用户操作，将操作后的页面效果进行截屏保存，此外保留第一次成功运行后的所有截屏图片，回归测试时可利用新旧截屏图片对比是否一致找出问题。

### 目录及文件介绍
- compared ：存放前后图片对比结果生成的图片
- imgs ：存放页面截图
    - first-test ：存放第一次运行测试脚本(main.js)所产生的的页面截图
- src：存放核心源码
    - test ：存放测试脚本
- compare.js ：存放基于resemblejs的图片对比脚本
- main.js ：入口文件
- runDiff.js ：执行图片对比
